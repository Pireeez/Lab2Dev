Elemento <h1>

Descrição:Define um cabeçalho de nível 1.


Atributos principais: id, class, style


Quando utilizar:Para destacar o título principal de uma página.


Exemplo:<h1>Bem-vindo à Enciclopédia</h1>

Elemento <p>

Descrição: Representa um parágrafo de texto.


Atributos principais:id, class, style, lang


Quando utilizar: Para exibir blocos de texto.


Exemplo:<p>Esse é um parágrafo explicativo.</p>

Elemento <a>

Descrição: Cria um link clicável para outra página ou recurso.


Atributos principais:href, target, rel


Quando utilizar: Sempre que quiser direcionar o usuário para outro local.


<a href="https://developer.mozilla.org" target="_blank">Visite a MDN</a>
