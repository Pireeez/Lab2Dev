Elemento <p>

Descrição: Representa um parágrafo de texto.


Atributos principais:id, class, style, lang


Quando utilizar: Para exibir blocos de texto.


Exemplo:<p>Esse é um parágrafo explicativo.</p>

Elemento <a>

Descrição: Cria um link clicável para outra página ou recurso.


Atributos principais:href, target, rel


Quando utilizar: Sempre que quiser direcionar o usuário para outro local.


<a href="https://developer.mozilla.org" target="_blank">Visite a MDN</a>

Elemento <h1>

Descrição: Representa o título de nível 1 (mais importante).


Atributos principais: id, class, style


Quando utilizar: Para destacar o título principal de uma seção ou página.


Exemplo: <h1>Título Principal</h1>

Elemento <img>

Descrição: Insere uma imagem na página.


Atributos principais: src, alt, width, height, id, class


Quando utilizar: Para exibir imagens no conteúdo.


Exemplo: <img src="imagem.jpg" alt="Descrição da imagem">

Elemento <ul>

Descrição: Cria uma lista não ordenada (com marcadores).


Atributos principais: id, class, style


Quando utilizar: Para listar itens sem ordem específica.


Exemplo: <ul><li>Item 1</li></ul>

Elemento <ol>

Descrição: Cria uma lista ordenada (numerada).


Atributos principais: type, start, reversed, id, class


Quando utilizar: Para listar itens em ordem sequencial.


Exemplo: <ol><li>Primeiro item</li></ol>

Elemento <input>

Descrição: Campo de entrada de dados.


Atributos principais: type, name, placeholder, value, id


Quando utilizar: Para coletar dados do usuário.


Exemplo: <input type="text" placeholder="Digite aqui">

Elemento <form>

Descrição: Agrupa campos de entrada e envia dados.


Atributos principais: action, method, id, class


Quando utilizar: Para criar formulários de envio.


Exemplo: <form action="/enviar" method="post"></form>

Elemento <button>

Descrição: Cria um botão clicável.


Atributos principais: type, onclick, id, class


Quando utilizar: Para ações, envios de formulário ou interações.


Exemplo: <button type="submit">Enviar</button>

Elemento <div>

Descrição: Container genérico para agrupar conteúdos.


Atributos principais: id, class, style


Quando utilizar: Para organizar a estrutura da página.


Exemplo: <div class="container">Conteúdo aqui</div>

inherit

Descrição:Faz com que uma propriedade herde o valor do seu elemento pai, mesmo que essa propriedade normalmente não seja herdada por padrão.

Quando usar:Quando você quer forçar a herança de uma propriedade específica.

Exemplo de uso:

div {
 border: inherit;
}

initial

Descrição: Define a propriedade com seu valor padrão do CSS, ignorando valores herdados ou definidos anteriormente.

Quando usar: Quando você quer remover completamente qualquer estilo aplicado e voltar ao comportamento padrão do navegador.

Exemplo de uso:

p {
 margin: initial;
}

unset

Descrição: Remove qualquer valor aplicado: se a propriedade for herdável, ela será herdada; caso contrário, será definida com seu valor padrão.

Quando usar: Quando você quer que a propriedade se comporte como se não tivesse sido definida — herdando ou voltando ao padrão dependendo do caso.

Exemplo de uso:

span {
 font-weight: unset;
}

revert

Descrição: Reverte a propriedade para o valor aplicado por estilos do navegador (user-agent) ou folha de estilo anterior.

Quando usar: Quando você quer ignorar regras personalizadas e voltar ao estilo "original" do navegador ou regras anteriores da cascata.

Exemplo de uso:

a {
 text-decoration: revert;
}

Sinalizador !important

Descrição:O sinalizador !important é usado para forçar uma regra CSS a se sobrepor a outras, mesmo que elas tenham maior especificidade.


Quando usar: Use  !important apenas quando necessário, como última alternativa para resolver conflitos de estilo. Evite o uso excessivo, pois ele pode dificultar a manutenção do código.


Exemplo de utilização:


HTML:

<div class="box-important">Texto com cor forçada!</div>

CSS:

.box-important {
 color: red !important;
}
